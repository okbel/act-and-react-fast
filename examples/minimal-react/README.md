# Minimal React Example

This is a minimal react example
Contains: 
  - Router
  - Inline Styles
  - ES6


## Get Started

`yarn` 
> This installs all the dependencies

`yarn dev`
> Runs the dev enviroment

`yarn build`
> Builds bundle for production

`yarn start`
> Runs express server (for production)
